# Installation

You can run agr without installing it, or install it globally.

## Run without installing

```bash
uvx agr add kasperjunge/hello-world
```

## Install with pip

```bash
pip install agr
```

Verify the installation:

```bash
agr --help
```

## Update

Upgrade with pip:

```bash
pip install --upgrade agr
```
